DQN Architecture
State Space & Action Space
STATE VECTOR (per process × N processes)
s[0] = remaining burst time (normalized)
s[1] = priority (0-9)
s[2] = current wait time (normalized)
s[3] = arrival time (normalized)
ACTION SPACE
a ∈ {0, 1, ..., N-1} = index of process to schedule
REWARD FUNCTION
r = -wait_penalty + completion_bonus
wait_penalty = Σ wait_time / max_burst
completion_bonus = +10 per process done
BELLMAN UPDATE
Q(s,a) ← Q(s,a) + α[r + γ·maxQ(s',a') - Q(s,a)]
NETWORK LAYERS
Input: state_size → Hidden1: 128 → Hidden2: 64 → Output: n_actions
Activation: ReLU | Optimizer: Adam | Loss: MSE
