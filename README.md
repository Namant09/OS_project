 RL-Based CPU Scheduler

Project Description
This project implements a CPU scheduling system using Reinforcement Learning (Deep Q-Network). The agent learns to select the best process to minimize waiting time and improve CPU utilization.

Features
- Simulation of CPU scheduling
- Algorithms: FCFS, SJF, Round Robin, RL-based
- Gantt Chart visualization
- Training of RL agent
- Performance comparison

Technologies Used
- HTML, CSS, JavaScript
- Reinforcement Learning (DQN)

 How to Run
Download the HTML file and open it in any web browser.
